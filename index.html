  • structural changes to the website
  • a cohesive colour scheme that is inviting and modern
  • user friendly layout of content
  • simpler navigation
  • updating old code to newer versions

Currently, your website makes good use of displaying the content it has in easy to read pages that allows users to focus solely on the content in front of them. You also have a search function that is incredibly beneficial to the end user who seeks to find exactly what they are looking for in a timely manner.

The changes I would like to take to improve upon the good aspects of your website would entail a few things.

Code update

Aligning content correctly so it becomes easier to consume and understand for the end user. Combining certain pages with relevant information but creating breaks to indicate a new section of content. Organising text and images correctly throughout multiple pages so that there will be a template to follow for future pages that are content heavy. Moving the search bar into the header itself will allow it to move with the page and not move onto content when screen sizes change.

Colour scheme changes

Altering the background colour and creating a greater contrast between the colours used makes the website more visually appealing while keeping the historic theme throughout the website. Complementary tones would be chosen to ensure the website continues to employ a certain feel about it while being modern and clean.

Intuitive navigation

Creating an intuitive navigation bar with drop down menus will make it easier for the end user to find the pages they want to visit. To do this I will combine some similar pages and create smaller drop down menus for topics such as the year of battle. This will ensure a better user experience when navigating the entire website.
